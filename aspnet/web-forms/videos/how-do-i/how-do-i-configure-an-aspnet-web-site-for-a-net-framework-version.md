<a name="how-do-i-configure-an-aspnet-web-site-for-a-net-framework-version"></a>[Nasıl stop yaparım] .NET Framework sürümü için bir ASP.NET Web sitesi yapılandırması
====================
tarafından [Chris Pels](https://twitter.com/chrispels)

Bu video Chris Pels ASP.NET/.NET Framework belirli bir sürümünü kullanmak için bir ASP.NET web sitesinin nasıl yapılandırılacağı gösterir. Önce bir web sitesi Internet Information Server (IIS) komut dosyası eşlemesini görüntüleyerek ilişkili olduğu .NET Framework'ün hangi sürümünün belirlemek bkz. Betik eşlemesi nasıl IIS'de kök ve ardından özel web siteleri ile ilişkili bakın. Sonraki ASP.NET IIS Kayıt Aracı hakkında bilgi edinin (aspnet\_regiis.exe) IIS veya bir ASP.NET web sitesini belirli bir .NET Framework sürümünü kullanmak üzere yapılandırmak için kullanılır. Bu bilgi, tek bir web sitesi IIS varsayılan olandan farklı bir .NET Framework sürümünü kullanmak için nasıl yapılandırıldığını bkz kullanma. Son olarak, bir .NET Framework 2.0 ve diğer altında 1.1 altında çalıştığı aynı IIS altında iki farklı web sitelerine bakın.

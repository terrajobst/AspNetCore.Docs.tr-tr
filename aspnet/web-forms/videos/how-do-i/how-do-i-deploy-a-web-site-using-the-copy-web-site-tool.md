<a name="how-do-i-deploy-a-web-site-using-the-copy-web-site-tool"></a>[Nasıl stop yaparım] Kopyalama Web sitesi Aracı'nı kullanarak bir Web sitesi dağıtma
====================
tarafından [Chris Pels](https://twitter.com/chrispels)

Bu video Chris Pels Visual Studio kopyası Web sitesi Aracı'nı kullanarak bir web sitesinin nasıl dağıtılacağı gösterir. İlk uzak bir web sitesine bağlanmak ve bütün bir siteyi hedef konuma dağıtma konusunda bakın. Ardından, tüm kaynak ve hedef web sitesine eşitleme öğrenin. Ayrıca, kopyalama ve tek tek veya Web sitesini kopyalama aracı dosyaları arasındaki farklar durum göstergelerini nasıl sağladığını dahil olmak üzere dosya grupları eşitleme konusuna bakın. Nasıl bu işlemin bir parçası gördüğünüz gibi aracı durumu iki veya daha fazla dosya arasındaki farkları olduğunda kararı için dosya tarafından temelinde seçenekler sunar.

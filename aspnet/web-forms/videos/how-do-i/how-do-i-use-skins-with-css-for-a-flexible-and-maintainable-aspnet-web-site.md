<a name="how-do-i-use-skins-with-css-for-a-flexible-and-maintainable-aspnet-web-site"></a>[Nasıl stop yaparım] Dış CSS ile esnek ve sürdürülebilir ASP.NET Web sitesi için kullanın.
====================
tarafından [Chris Pels](https://twitter.com/chrispels)

Bu video Chris Pels, ASP.NET sunucu denetimleri için kaplamaları oluşturmak ve dış görünüm denetimlerin görünümünü denetlemek üzere CSS stillerinin birleştirme gösterilmektedir. İlk olarak, bir ASP.NET sunucu denetimi gibi bir web sitesi boyunca etiketleri görünümünü denetleyen bir etiket için bir dış görünüm oluşturmayı öğrenin. Ardından, CSS stilleri dış görünüm için görsel görünümünü tanımını soyut konusuna bakın. Bu işlem sırasında bir dış stil bilgilerini CSS stilleri sahip avantajları öğrenin. Daha sonra birden çok kaplamaları ve tek bir ASP.NET denetimi gibi belirli türde bir web sitesi boyunca etiketleri görünümünü standart hale getirmek için kullanılan bir etiket için karşılık gelen stiller nasıl oluşturulacağını bakın. Son olarak, birden çok kaplamaları ve genel bir HTML öğesi uygulama özelliği için bir CSS stil uygulamadan avantajları öğrenin.

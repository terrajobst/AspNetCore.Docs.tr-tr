<a name="jquery-cycle-294"></a>jQuery döngüsü 2.94
====================
Aşağıdaki jQuery döngüsü dosyaları bu CDN üzerinde barındırılan:

- http://AJAX.aspnetcdn.com/AJAX/JQuery.Cycle/2.94/JQuery.Cycle.all.js &mdash; jQuery döngüsü eklentisi'nın tam sürümünü. Tüm birden çok iki düzine özel etkiler ve tüm seçenekleri içerir.
- http://AJAX.aspnetcdn.com/AJAX/JQuery.Cycle/2.94/JQuery.Cycle.all.Min.js &mdash; yukarıdaki dosyanın küçültülmüş sürümü. Web sitesi performansını artırmak için bu sürümü üretim uygulamaları için kullanın.
- http://AJAX.aspnetcdn.com/AJAX/JQuery.Cycle/2.94/JQuery.Cycle.js &mdash; jQuery döngüsü eklentisi bu sürümü daha küçüktür ve yalnızca "yavaşça" geçiş etkisi içerir.
- http://AJAX.aspnetcdn.com/AJAX/JQuery.Cycle/2.94/JQuery.Cycle.Min.js &mdash; yukarıdaki dosyanın küçültülmüş sürümü. Web sitesi performansını artırmak için bu sürümü üretim uygulamaları için kullanın.
- http://AJAX.aspnetcdn.com/AJAX/JQuery.Cycle/2.94/JQuery.Cycle.Lite.js &mdash; yalnızca "yavaşça" geçişi olmadan tüm seçenekleri içeren jQuery döngüsü eklentisi bile daha küçük bir sürümü.
- http://AJAX.aspnetcdn.com/AJAX/JQuery.Cycle/2.94/JQuery.Cycle.Lite.Min.js &mdash; yukarıdaki dosyanın küçültülmüş sürümü. Web sitesi performansını artırmak için bu sürümü üretim uygulamaları için kullanın.

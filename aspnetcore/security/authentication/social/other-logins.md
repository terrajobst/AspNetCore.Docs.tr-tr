# <a name="short-survey-of-other-authentication-providers"></a>Diğer kimlik doğrulama sağlayıcıları kısa anket

<a name="security-authentication-other-logins"></a>

Tarafından [Rick Anderson](https://twitter.com/RickAndMSFT), [Pranav Rastogi](https://github.com/rustd), ve [Valeriy Novytskyy](https://github.com/01binary)

Burada bazı diğer ortak OAuth sağlayıcılar için yönergeler yukarı ayarlanır. Üçüncü taraf NuGet paketleri tarafından tutulan olanlar gibi [aspnet contrib](https://www.nuget.org/packages?q=owners%3Aaspnet-contrib+title%3AOAuth) ASP.NET Core ekibi tarafından gerçekleştirilen kimlik doğrulama sağlayıcıları tamamlamak için kullanılabilir.

* Ayarlanan **LinkedIn** oturum açın: [https://www.linkedin.com/developer/apps](https://www.linkedin.com/developer/apps). Bkz: [resmi adımları](https://developer.linkedin.com/docs/oauth2).

* Ayarlanan **Instagram** oturum açın: [https://www.instagram.com/developer/register/](https://www.instagram.com/developer/register/). Bkz: [resmi adımları](https://www.instagram.com/developer/authentication/).

* Ayarlanan **Reddit** oturum açın: [https://www.reddit.com/login?dest=https%3A%2F%2Fwww.reddit.com%2Fprefs%2Fapps](https://www.reddit.com/login?dest=https%3A%2F%2Fwww.reddit.com%2Fprefs%2Fapps). Bkz: [resmi adımları](https://github.com/reddit/reddit/wiki/OAuth2-Quick-Start-Example).

* Ayarlanan **Github** oturum açın: [https://github.com/login?return_to=https%3A%2F%2Fgithub.com%2Fsettings%2Fapplications%2Fnew](https://github.com/login?return_to=https%3A%2F%2Fgithub.com%2Fsettings%2Fapplications%2Fnew). Bkz: [resmi adımları](https://developer.github.com/v3/oauth/).

* Ayarlanan **Yahoo** oturum açın: [https://login.yahoo.com/config/login?src=devnet&.done=http%3A%2F%2Fdeveloper.yahoo.com%2Fapps%2Fcreate%2F](https://login.yahoo.com/config/login?src=devnet&.done=http%3A%2F%2Fdeveloper.yahoo.com%2Fapps%2Fcreate%2F). Bkz: [resmi adımları](https://developer.yahoo.com/bbauth/user.html).

* Ayarlanan **Tumblr** oturum açın: [https://www.tumblr.com/oauth/apps](https://www.tumblr.com/oauth/apps). Bkz: [resmi adımları](https://www.tumblr.com/docs/api/v2#auth).

* Ayarlanan **Pinterest** oturum açın: [https://www.pinterest.com/login/?next=http%3A%2F%2Fdevsite%2Fapps%2F](https://www.pinterest.com/login/?next=http%3A%2F%2Fdevsite%2Fapps%2F). Bkz: [resmi adımları](https://developers.pinterest.com/docs/api/overview/?).

* Ayarlanan **Pocket** oturum açın: [https://getpocket.com/developer/apps/new](https://getpocket.com/developer/apps/new). Bkz: [resmi adımları](https://getpocket.com/developer/docs/authentication).

* Ayarlanan **Flickr** oturum açın: [https://www.flickr.com/services/apps/create](https://www.flickr.com/services/apps/create). Bkz: [resmi adımları](https://www.flickr.com/services/api/auth.oauth.html).

* Ayarlanan **Dribble** oturum açın: [https://dribbble.com/signup](https://dribbble.com/signup). Bkz: [resmi adımları](http://developer.dribbble.com/v1/oauth/).

* Ayarlanan **Vimeo** oturum açın: [https://vimeo.com/join](https://vimeo.com/join). Bkz: [resmi adımları](https://developer.vimeo.com/api/authentication).

* Ayarlanan **SoundCloud** oturum açın: [https://soundcloud.com/you/apps/new](https://soundcloud.com/you/apps/new). Bkz: [resmi adımları](https://developers.soundcloud.com/blog/we-love-oauth-2).

* Ayarlanan **VK** oturum açın: [https://vk.com/apps?act=manage](https://vk.com/apps?act=manage). Bkz: [resmi adımları](https://vk.com/pages?oid=-17680044&p=Authorizing_Sites).

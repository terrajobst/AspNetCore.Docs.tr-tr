## <a name="authentication-library"></a><span data-ttu-id="c6696-107">Kimlik doğrulama kitaplığı</span><span class="sxs-lookup"><span data-stu-id="c6696-107">Authentication library</span></span>

* <span data-ttu-id="c6696-121">Belirteçler, varsayılan olarak bir saat, saldırı penceresini sınırlayan kısa bir yaşam süresine sahiptir.</span><span class="sxs-lookup"><span data-stu-id="c6696-121">Tokens have a short lifetime, one hour by default, which limits the attack window.</span></span> <span data-ttu-id="c6696-122">Belirteçler de herhangi bir zamanda iptal edilebilir.</span><span class="sxs-lookup"><span data-stu-id="c6696-122">Tokens can also be revoked at any time.</span></span>

* <span data-ttu-id="c6696-126">OAuth ve OıDC içeren belirteçler, uygulamanın güvenli olduğundan emin olmak için doğru şekilde davranan kullanıcı aracısına güvenmeyin.</span><span class="sxs-lookup"><span data-stu-id="c6696-126">Tokens with oAuth and OIDC don't rely on the user agent behaving correctly to ensure that the app is secure.</span></span>
* <span data-ttu-id="c6696-127">OAuth ve OıDC gibi belirteç tabanlı protokoller aynı güvenlik özellikleriyle barındırılan ve tek başına uygulamaların kimliğini doğrulamaya ve yetkilendirmeye olanak tanır.</span><span class="sxs-lookup"><span data-stu-id="c6696-127">Token-based protocols, such as oAuth and OIDC, allow for authenticating and authorizing hosted and standalone apps with the same set of security characteristics.</span></span>
